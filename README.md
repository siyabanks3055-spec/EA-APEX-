# ⚡ EA APEX - Trading Dashboard

AI-powered trading signals dashboard with technical analysis, AutoTrader, and MT5 integration.

## Features

- 📊 **Technical Analysis** - Advanced charting tools with multiple indicators
- 🤖 **AutoTrader** - Automated trading execution with AI-powered signals
- 📈 **MT5 Integration** - Seamless MetaTrader 5 connectivity
- 📱 **Progressive Web App** - Install as a native app on any device
- 🌙 **Dark/Light Theme** - Customizable interface
- 📡 **Real-time Signals** - Instant trading alerts and notifications
- 🔐 **Secure** - Enterprise-grade security for accounts

## Installation

### As a Web App
1. Visit the deployed URL
2. Click "Install App" button or use browser menu
3. Add to your home screen
